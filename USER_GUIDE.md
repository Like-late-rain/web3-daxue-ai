## 准备工作

### 1. 安装 MetaMask

1. 访问 [MetaMask 官网](https://metamask.io/)
2. 下载并安装浏览器扩展
3. 创建钱包或导入现有钱包
4. 妥善保管助记词

### 2. 添加 Sepolia 测试网

1. 打开 MetaMask
2. 点击网络下拉菜单
3. 点击"添加网络"
4. 选择 "Sepolia" 测试网络

### 3. 获取测试 ETH

访问以下任一水龙头获取免费的 Sepolia ETH：

- [Alchemy Sepolia Faucet](https://sepoliafaucet.com/)
- [Infura Sepolia Faucet](https://www.infura.io/faucet/sepolia)
- [Chainlink Sepolia Faucet](https://faucets.chain.link/sepolia)

## 获取 YCT 代币

### 方法一：兑换中心

1. 点击导航栏的"兑换中心"
2. 选择"购买 YCT"模式
3. 输入要支付的 ETH 数量
4. 系统自动计算可获得的 YCT 数量
   - 兑换比例：0.0001 ETH = 1 YCT
   - 例如：0.1 ETH = 1000 YCT
5. 点击"购买 YCT"
6. 在 MetaMask 中确认交易
7. 等待交易确认

### 方法二：直接与合约交互

如果您熟悉智能合约，可以直接调用 `YCToken` 合约的 `buyTokens()` 函数。

## 购买课程

1. 在课程详情页，点击"购买课程"按钮
2. **第一步**：授权 YCT 代币
   - 在 MetaMask 中确认授权交易
   - 这允许课程合约使用您的 YCT
3. **第二步**：购买课程
   - 授权成功后，自动进行购买
   - 在 MetaMask 中确认购买交易
4. 等待交易确认
5. 购买成功后，课程会出现在"个人中心"的"我的课程"中

### 注意事项

- 确保 YCT 余额充足
- 每个课程只能购买一次
- 不能购买自己创建的课程
- 购买后平台会收取 5% 手续费
  - 95% 支付给教师
  - 5% 支付给平台

## 创建课程

### 成为教师

任何钱包地址都可以创建课程，无需额外申请。

### 创建步骤

1. 点击导航栏的"创建课程"
2. 填写课程信息：

   **必填字段：**
   - 课程标题：简洁明确的标题
   - 课程描述：详细介绍课程内容
   - 课程价格：以 YCT 为单位

   **可选字段：**
   - 封面 URL：上传到 Cloudflare R2 的图片链接

3. 点击"创建课程"
4. 在 MetaMask 中确认交易
5. 等待交易确认
6. 创建成功后，课程会出现在市场中

### 上传课程封面（可选）

如果您配置了 Cloudflare R2：

1. 登录 Cloudflare Dashboard
2. 进入 R2 Bucket
3. 上传图片文件
4. 复制公开访问 URL
5. 粘贴到"封面 URL"字段

**推荐图片规格：**
- 格式：JPG、PNG
- 尺寸：1200x630px
- 大小：< 1MB

## 常见问题

### Q1: 为什么交易失败？

**可能原因：**
- Gas 费用不足
- 网络拥堵
- YCT 余额不足
- 未授权代币使用

**解决方法：**
- 确保钱包有足够的 ETH 支付 Gas
- 稍后重试
- 检查 YCT 余额
- 重新授权代币

### Q2: 如何查看交易记录？

1. 在 MetaMask 中查看"活动"
2. 访问 [Sepolia Etherscan](https://sepolia.etherscan.io/)
3. 输入您的钱包地址查询

### Q3: 课程购买后在哪里学习？

目前平台主要展示课程信息和管理购买权限。实际课程内容的托管和学习功能将在后续版本中添加。

### Q4: 可以退款吗？

由于区块链交易的不可逆性，课程购买后无法退款。购买前请仔细确认。

### Q5: 如何联系教师？

目前只显示教师的钱包地址。未来版本可能会添加消息功能。

## 安全提示

⚠️ **重要安全建议：**

1. **保护私钥**
   - 永远不要分享私钥或助记词
   - 不要在任何网站输入助记词

2. **验证网站**
   - 确认访问的是官方网站
   - 检查 URL 地址

3. **确认交易**
   - 仔细检查 MetaMask 中的交易详情
   - 确认合约地址正确

4. **测试网络**
   - 本项目运行在 Sepolia 测试网
   - 不要发送主网 ETH 或代币

5. **备份数据**
   - 保存好钱包助记词
   - 记录重要的交易哈希
