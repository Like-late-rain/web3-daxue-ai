# 贡献指南

感谢您考虑为 Web3 涂山大学项目做出贡献！

## 如何贡献

### 报告 Bug

1. 检查 [Issues](https://github.com/YOUR_USERNAME/web3-daxue/issues) 确认问题未被报告
2. 创建新 Issue，包含以下信息：
   - Bug 描述
   - 复现步骤
   - 预期行为
   - 实际行为
   - 截图（如适用）
   - 环境信息（浏览器、操作系统等）

### 提交功能请求

1. 检查是否已有类似的功能请求
2. 创建新 Issue，详细描述功能需求和使用场景

### 提交代码

1. Fork 本仓库
2. 创建新分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 代码规范

- 使用 TypeScript
- 遵循 ESLint 规则
- 编写清晰的提交信息
- 添加必要的注释和文档
- 确保所有测试通过

### 智能合约更改

如果您要修改智能合约：

1. 添加完整的测试用例
2. 确保通过所有现有测试
3. 考虑安全性和 Gas 优化
4. 更新相关文档

### 前端更改

1. 确保响应式设计
2. 测试不同浏览器的兼容性
3. 优化性能和加载速度
4. 保持 UI/UX 一致性

## 行为准则

- 尊重所有贡献者
- 欢迎建设性的反馈
- 专注于对项目最有利的方面
- 展现同理心和善意

## 问题？

如有任何疑问，请随时创建 Issue 或联系维护者。

感谢您的贡献！
